# ⚽ Champions League Players API

API REST simples para gerenciamento de jogadores da UEFA Champions League, permitindo realizar operações básicas de CRUD (Create, Read, Update, Delete).

O projeto foi desenvolvido para fins de aprendizado e prática de conceitos de APIs REST, manipulação de dados e organização de rotas.

# 🚀 Funcionalidades

A API permite:

- 📄 Listar todos os jogadores

- 🔎 Buscar um jogador específico

- ➕ Adicionar um novo jogador

- ✏️ Atualizar informações de um jogador

- ❌ Remover um jogador

# 🛠️ Tecnologias Utilizadas

Node.js

Express

JavaScript / TypeScript

JSON para armazenamento de dados
